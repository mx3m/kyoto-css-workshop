FAQ

  • Do I need any prior knowledge of HTML & CSS?

    Not necessarily but it's good if you already know how to play with the Chrome Dev Tools and have some very basic understanding of HTML. If I can get a big enough group of beginners, I will arrange for a 100% beginners series of events so feel free to shoot me an email.

  • Where is it?

    Each event's location will be shared on Facebook. It will mostly be around the Shijo-Karasuma area in Kyoto.

  • How much is it?

    It mostly depends on the location. The first event will be ¥1000/person.
